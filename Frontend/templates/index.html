About the Project

The Distributed Event Booking System combines real-time synchronization, concurrency control, and consistent replication to simulate how modern ticketing platforms maintain reliability and speed across multiple servers.

Developed with Flask for backend coordination and HTML/CSS/JS for frontend interaction, it demonstrates essential distributed system concepts in a hands-on, interactive way.
